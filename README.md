# 🎓 OrienGO – Smart Career Orientation Platform

**OrienGO** is a full-stack web platform designed to guide students in choosing their future careers based on the **RIASEC personality model** and **Artificial Intelligence (AI)**.

---

## 🧠 Project Overview

OrienGO combines psychometric evaluation, AI-powered recommendations, and an intuitive dashboard to provide:
- 🧪 RIASEC career tests
- 🧠 AI-based career suggestions
- 📊 Dashboards for students and coaches
- 🧑‍🏫 Coach-student management
- 🔔 Notifications & profile settings
- 🌐 Multilingual support (EN/FR)

---

## 🏗️ Tech Stack

| Layer        | Technology                        |
|--------------|-----------------------------------|
| Frontend     | Angular 18, Tailwind CSS, i18n    |
| Backend      | Spring Boot (REST API)            |
| AI Module    | Python (Flask, scikit-learn)      |
| Database     | PostgreSQL                        |
| DevOps       | GitHub, Azure DevOps (sprints)    |

---

## 🧩 Main Features

### 👨‍🎓 Student
- Take the RIASEC test
- View personalized job recommendations
- Browse job cards with details
- Update profile & settings

### 🧑‍🏫 Coach
- Manage student profiles
- Track student scores & matches
- Dashboard with insights
- Moderate & share results

### ⚙️ Admin (Planned)
- User role management
- Analytics and access control
